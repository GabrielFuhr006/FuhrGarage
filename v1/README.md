# FuhrGarage 🚗✨

Landing page institucional desenvolvida para a marca FuhrGarage, especializada em estética automotiva e detalhamento premium.

## 🧼 Sobre o Projeto
Este site foi criado com o objetivo de apresentar os serviços de lavagem detalhada, descontaminação e cuidados automotivos, fortalecendo a presença digital da marca.

## 🚀 Serviços Oferecidos
- Lavagem de Manutenção
- Lavagem Detalhada
- Descontaminação Ferrosa
- Higienização Interna
- Cuidado com pintura e acabamento

## 💻 Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo
Desenvolver uma presença digital profissional para atrair clientes, divulgar serviços e transmitir qualidade, confiança e cuidado automotivo.

## 📈 Próximas Melhorias
- Responsividade para mobile
- Integração com WhatsApp
- Otimização para SEO
- Deploy online
